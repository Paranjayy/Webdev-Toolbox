# Antigravity Dev Vault: God Build Edition 🔓🚀

The ultimate toolkit for reverse engineers, web scrapers, and "vibecoding" speed-demons. Built to bridge the gap between the browser and the LLM.

## 🌟 Core Features

### 🧩 Workspaces & Lifecycle
- **Extension Reloader**: Hot-reload unpacked extensions and their target tabs in one click.
- **Domain Reloader**: Refresh every open tab on the current domain at once.
- **Sentinel Monitor**: specialized error tracking (e.g., Boot.dev Sentinel) + Global DOM Error catching.

### ⚡ Intelligence & Extraction
- **⚡ GIGASNAP**: The master command. Captures **URL, Stack Analysis (React/Next/Tailwind), Performance Metrics, Browser Metadata, Storage, Cookies,** and a **Token-Optimized Clean DOM**. Output is wrapped in a "God Prompt" ready for AI analysis.
- **AI Task Annotator**: Select multiple UI elements, write feedback for each, and copy a Markdown list of Tasks+Selectors for your LLM.
- **JSON Hunter**: Digs up hidden `application/ld+json` tags and window-level framework data (Next, YT, etc).
- **Theme Extractor**: Sucks out every CSS Variable (`--`) and color palette from the page.

### 🛠️ Reverse Engineering Toolkit
- **Network Interceptor**: Styled hooks for `fetch` and `XHR` that log payloads directly to your console.
- **Console Watcher**: Proxies system logs with visual tags for easier filtering.
- **Selector Picker**: Click any element to get its most unique CSS selector for your scrapers.
- **Page Freezer**: Pause the world in 3 seconds to inspect hover states and popups.
- **Lazy Load Trigger**: Smoothly scrolls to the bottom to force infinite-scroll content to spawn.

### 🎨 Vibes
- **Cyberpunk Vibe**: Toggle a neon-noir Matrix theme on any website.
- **Styled Console**: High-fidelity terminal-style feedback.

## 🚀 How to use GIGASNAP
Just hit the **⚡ GIGASNAP** button in the header. The extension will gather the entire site's context, clean the DOM to save you **70% in LLM tokens**, and prepare a perfect prompt for ChatGPT/Claude/Gemini.
