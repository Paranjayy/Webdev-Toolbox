# GIGASNAP Dev Vault: God Build Edition 🔓🚀

The ultimate orchestration utility for reverse engineers, web scrapers, and "vibecoding" speed-demons. Built to bridge the gap between the browser and the LLM with high-fidelity intelligence gathering.

## 🌟 Core Features

### ⚡ Intelligence & Extraction (GIGASNAP)
- **⚡ GIGASNAP**: The master command. Captures **URL, Stack Analysis, Performance, Storage, Cookies,** and a **Token-Optimized Clean DOM**. Supports **Shadow DOM** recursion.
- **Vibe Snap**: Bundles the entire page context into a single Markdown prompt for instant AI briefing.
- **Global Variable Hunter**: Scans `window` for hidden config/data objects (Next.js, YT, etc.).
- **Theme Extractor**: Sucks out every CSS Variable and color palette for design system analysis.

### 🕵️ Privacy & Scrambling (New!)
- **Privacy Shield**: Auto-redacts Emails and Phone Numbers from the visible page for safe sharing.
- **Ghost Mode**: Blurs all images, avatars, and videos with one click.
- **Manual Scrambler**: Find and replace any text mentions on the page manually.

### 🕹️ Hacker & Dev Fun
- **Gravity Mode**: Modify the gravity constant and watch DOM elements fall.
- **Text Exploder**: Break words into individual characters and scatter them.
- **Selector Picker**: Copy unique CSS selectors instantly.
- **Auto Scroll**: Automatically trigger lazy-loading by scrolling to bottom.

### 🧩 Workspaces & Lifecycle
- **Extension Reloader**: Hot-reload unpacked extensions and target tabs in one click.
- **Domain Reloader**: Refresh every open tab on the current domain at once.
- **Vault Monitor**: Specialized error tracking + Global DOM/Sentinel Error catching.

### 📡 Runtime Interceptors
- **Network Interceptor**: Styled hooks for `fetch` and `XHR`.
- **WS Sniffer**: Hook WebSocket constructors to log stream data.
- **Page Freezer**: Pause the world in 3 seconds to inspect hover states.

## ⚡ GIGASNAP Deep-Dive

GIGASNAP is designed to provide the absolute maximum context to an LLM while staying within token limits. Every snap generates a structured JSON object containing:

- **Stack Detection**: Auto-detects React, Next.js, Vue, Tailwind, Framer Motion, Radix UI, and more.
- **Design Tokens**: Samples the page to extract active CSS color palettes and font families.
- **Performance Intelligence**: Captures TTFB, Load Time, and Transfer Size to help debug slow interactions.
- **Global Data Hunter**: Deep-scans the `window` object for hidden framework data (like `__NEXT_DATA__` or Apollo state).
- **Security & Privacy**: 
    - Auto-redacts Emails and Phone numbers from the visible DOM.
    - Captures non-HttpOnly cookies and localStorage for session debugging.
- **Token-Optimized DOM**: A recursive cleaner that:
    - Pierces **Shadow DOM** boundaries.
    - Strips bloat (scripts, styles, heavy SVG paths).
    - Preserves semantic IDs and Classes for layout understanding.

## 🚀 How to use
1. **Right-Click** anywhere on a page and select **⚡ GIGASNAP** to copy the full context to your clipboard.
2. Open the **Vault Popup** to access specialized tools like **Ghost Mode** or **Manual Scrambler**.
3. Paste the clipboard into ChatGPT/Claude/Gemini to start debugging with "God-like" context.
