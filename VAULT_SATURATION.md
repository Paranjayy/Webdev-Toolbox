### Intelligence/AI Features
- **Network Vault**: Persistent 100-event traffic buffer, WebSocket sniffing
- **Auto-Snap on 5xx**: Automatically captures DOM state on server errors
- **$v Global Debugger**: `$v.scan()`, `$v.rip(el)`, `$v.help()` in any tab console
- **Snapshot History**: Last 5 captures stored, viewable in popup
- **Visual Diff**: Structural comparison between any 2 snapshots
- **Anti-Slop Scan**: Impeccable rules (purple gradients, overused fonts, gradient text, cardocalypse, zero contrast, pill buttons, side-tab cards)
- **Deep Rip Blueprint**: Full extension manifest/permissions capture for AI replication
