# 🚀 The Vault Extension Roadmap

Unified high-fidelity dashboard for extension orchestration, reverse ops, and AI-assisted debugging.

## 🛠 Currently Integrated / Stable
- **The Vault Dashboard**: Unified management of all local and store extensions.
- **7TV Search Pro (Raycast)**: GQL-powered emote discovery with "Paste-on-Enter".
- **SVGL Search Pro (Raycast)**: Grid-first high-fidelity icon search.
- **Pulse Harvest Pro**: Data extraction for LLM context (ChatGPT, Gemini, Claude).
- **Omniscient Chess**: Stockfish-powered analytics overlay.
- **Sentinel**: Boot.dev automated scraper and error monitor.

## 🏗 Planned Merges & Consolidation
- [ ] **Omni-Exporter Unified**: Merge IG, Twitter, and Valorant scrapers into a single high-performance engine.
- [ ] **Knowledge Bank**: Integrate Obsidian Clipper directly into The Vault's dashboard for one-click knowledge hoarding.
- [ ] **DOM Cleaner Pro**: Port the standalone GigaSnap logic into a background service for persistent site state tracking.
- [ ] **Media Hoardr**: Combine YouTube Screenshot, Twitch VOD Downloader, and Image Downloader into a unified "Media Center" extension.

## 🧪 Reverse Ops & Advanced Tools
- [ ] **Wappalyzer Deep-Link**: Automate tech-stack detection and link it to relevant GitHub search queries for reverse engineering.
- [ ] **Auto-Solver Hub**: Create a modular API for solver extensions (Matiks, Duolingo) to use a shared LLM backend.
- [ ] **ShadowRoot Snap**: Add support for deep Shadow DOM traversal in the Snapshot tool.

## 🎨 Design System
- [ ] **Unslopified UI**: Replace all default extension popups with The Vault's premium glassmorphism theme.
